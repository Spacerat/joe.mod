
Joe's Blitzmax Modules


--RIFF.mod

Provides an interface for reading and writing to RIFF/IFF files.

--Snarl.mod

Provides an interface to the Snarl notification system for Windows.

--Threadedio.mod

Provides a standardised system for thread safe IO.

--Colour.mod

Provides a useful class for storing, converting and creating colour data.

--LogprintStream.mod

Provides a simple, pluggable utility for automatically writing all StandardIO output to a file.

--Advtext.mod

Provides a function for drawing text wrapped to a set width, and formatted with different colours/opacities.

--Tiles.mod

A tile engine, including a map writer/loader.
